## Roles & Process

| Term | Meaning |
|------|---------|
| **PM** | Project Manager |
| **PE** | Product Engineer |
| **QA** | Quality Assurance |
| **BA** | Business Analyst |
| **SDR** | Sales Development Representative |
| **Tech Lead** | Senior developer who leads platform-specific technical decisions |
| **SOW** | Statement of Work (contract document) |
| **PRD** | Product Requirements Document |
| **Sprint** | 2-week development cycle |
| **Refinement** | Pre-sprint session where devs review wireframes and PRD |
| **Handoff** | Transition of completed project from dev team to client |
| **3-Tier Proposal** | Pricing format: Basic (execute), Collaborative (recommend), Premium (lead) |
| **Price Anchoring** | Presenting highest tier first to anchor expectations |

## Content & Marketing

## Project Types

| Term | Meaning |
|------|---------|
| **MVP** | Minimum Viable Product — first functional version of a product |
| **SaaS** | Software as a Service — subscription-based software product |
